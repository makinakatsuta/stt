操作方法と公式ルール

🎮 操作キー

  • 左矢印キー (←) または Aキー: ラケットを左に移動
  • 右矢印キー (→) または Dキー: ラケットを右に移動
  • スペースキー (Space): アクションの決定 (サーブの構え、レシーブの返事「はい」、サーブ打球、ラリー中の打ち返し)
  • エスケープキー (Esc): プレイの中断

🏓 サウンドテーブルテニス (STT) とは？

視覚障害者向けの卓球で、ボールの中に金属球が4個入っており、転がると「シャラシャラ」と音が鳴ります。ネットの下（テーブル面から4.2cmの隙間）を通してボールを交互に転がし合い、得点を競います。

📜 試合のルール・流れ

  1. 審判の「プレー」宣告でスタートします。
  2. サーブの準備: サーバーはボールを静止させ、準備ができたらスペースキーを押して「いきます」と発声します（10秒以内）。
  3. レシーブの返事: レシーバーは「いきます」の後、スペースキーを押して「はい」と返答します（5秒以内）。
  4. サーブ: レシーバーが「はい」と答えてから、サーバーは5秒以内にスペースキーでサーブを打ちます。
  5. ラリー: 交互に打ち合います。ボールがパドルに近づいたタイミングでスペースキーを押して打ち返します。
  6. 得点判定: 打球が相手の守備コート内で「停止」するか、エンドフレーム（奥の壁）の内側面に触れて「セーフ（アウトにならずコート内に留まる）」になれば得点になります。
