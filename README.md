# サウンドテーブルテニス (STT) オンライン

サウンドテーブルテニス (STT) は、視覚障害者のために考案された卓球競技です。
このプロジェクトは、STTの公式ルールに準拠し、**視覚障害者（スクリーンリーダー利用者）も健常者も共に同じ条件で対戦できる**ことをコンセプトとした、GoバックエンドおよびWeb Audio API搭載のリアルタイムブラウザゲームです。

---

## 🌟 主な特徴

1. **Web Audio API による3D立体音響システム**
   * ボールの左右の移動をステレオパンニングで、奥行き（相手コート、ネット、自分コート）を音量と音質（ローパスフィルターによるこもり具合）の変化でシミュレートします。
   * 金属球が4個入ったSTT公式ボール特有の「シャラシャラ」という転がり音をオシレーターとノイズを用いてリアルタイムに合成します。
   * 木製ラケットで打球した時の「コン」、フレームに当たった時の「カツ」、ネットに当たった時の「ポス」など、素材感のある衝突音を再現しています。

2. **完全アクセシビリティ（スクリーンリーダー＆音声ガイド対応）**
   * Web Speech API (`SpeechSynthesis`) による自動音声案内（実況）を搭載。審判のコールや、サーブ前の掛け合い（「いきます」「はい」）をすべて音声で実況します。
   * スクリーンリーダーの操作を妨げないセマンティックなHTML構造とARIA属性設計。
   * キーボードの最小限のキー（左右矢印キーとスペースキー）だけで、メニュー操作から対戦プレイまで完結します。

3. **Go ＋ WebSocket によるリアルタイムオンライン対戦**
   * 1台のPCでのCPU対戦に加え、GoバックエンドによるWebSocket通信を利用した2人対戦（オンラインルーム）に対応。
   * ゲームステート（サーブ制限時間、デュース判定、得点処理）はGo側で厳密に判定・同期されるため、ラグがあっても一貫したゲーム進行が可能です。

4. **プレミアムな視覚デザイン（健常者向け）**
   * ダークモードを基調としたネオン・サイバーパンク調の美しいグラフィック。
   * ボールの軌跡や、衝突時にコート上に広がる「音の波紋（パルス）」を可視化。音の動きを視覚的にも体感できます。

## 🎮 遊び方・操作方法

### 基本操作
* **左矢印キー (`ArrowLeft`)** / **Aキー**: ラケットを左に移動
* **右矢印キー (`ArrowRight`)** / **Dキー**: ラケットを右に移動
* **スペースキー (`Space`)**: アクション（サーブの準備、相手への返答、メニュー決定など）
* **エスケープキー (`Escape`)**: ゲームの一時停止 / メニューに戻る

### STT公式ルールに基づくプレイフロー
1. **主審の「プレー」宣告**でラリーの準備が始まります。
2. **サーバー**: スペースキーを押してボールを静止させ、準備ができたら「いきます」と発声します。（10秒以内）
3. **レシーバー**: サーバーの「いきます」に対してスペースキーで「はい」と返答します。（5秒以内）
4. **サーバー**: 返答から5秒以内にスペースキーでサーブを打ちます。（5秒以内）
5. **ラリー開始**: ボールがネットの下（テーブル面上4.2cmの隙間）を通って転がってくるので、音を頼りにラケットを動かして打ち返します。
6. **得点判定**: ボールが相手の守備コート内で「停止」するか、エンドフレーム（奥の壁）の内側に当たって「セーフ」になれば得点です。フレームを越えて飛び出たり（アウト）、ネットに引っかかったり、ラケット以外の場所に当たると相手の得点になります。

## 🚀 起動方法

### 方法1：実行ファイルから起動（推奨・最も簡単）
1. プロジェクトルートにある `stt.exe` をダブルクリックして実行します。
2. サーバーが起動し、自動的にデフォルトのブラウザで `http://localhost:8080` が開きます。

### 方法2：Goソースコードから起動（開発者向け）
1. ターミナルでプロジェクトディレクトリに移動します。
   ```bash
   cd stt
   ```
2. 以下のコマンドでサーバーを起動します。
   ```bash
   go run main.go
   ```
3. サーバーが起動すると、自動的にブラウザで `http://localhost:8080` が開きます。
   * もしブラウザが自動で開かない場合は、手動で `http://localhost:8080` にアクセスしてください。
   * ※オンライン対戦をテストする場合は、2つのブラウザウィンドウから上記URLにアクセスし、「オンライン対戦」を選択してください。

## 🌐 別のパソコンと対戦する場合の手順（ローカルネットワーク対戦）

同じWi-Fi（ローカルネットワーク）に接続された別のPCやタブレット・スマートフォンと対戦することができます。

### 1. サーバーPC（親機）の準備
1. 親機となるPCで `stt.exe` をダブルクリックして起動します。
2. 親機PCのローカルIPアドレス（IPv4）を調べます。
   * **Windowsの場合**: コマンドプロンプトを開き、`ipconfig` と入力して実行します。「IPv4 アドレス」の値を確認します（例: `192.168.1.15`）。
   * ※Windowsのセキュリティ警告（ファイアウォール）が出た場合は、ネットワーク上での通信を許可してください。

### 2. 対戦相手PC（子機）の接続
1. 子機PC（またはスマートフォンなど）を、親機と同じWi-Fiネットワークに接続します。
2. 子機のWebブラウザを開き、アドレスバーに **「親機PCのIPアドレス:8080」** を入力してアクセスします。
   * 例: `http://192.168.1.15:8080`
3. 画面が表示されたら、「オンライン対戦」を選択します。

### 3. ルームに入って対戦開始
1. 親機・子機の双方で、ルームIDを**空欄のまま**「ルームに入る」ボタンを押します。（特定の相手とだけ遊びたい場合は、両方に `maki` など**全く同じルームID**を入力してボタンを押してください。）
2. これで自動的にマッチングし、対戦が始まります。
